Make your product search rapid and easier with ajax search!

== Description ==

This plugin allow you to search in realtime mode through your products.
Realtime search start after you type at least three characters in the search input.

Woocommerce ajax search can be activated in two ways:

*   by typing a shortcode "[ns-search-products-free]"
*   by adding a widget "NS Ajax Product Search"

Use this plugin is very easy and fast!

For more information about our plugins, visit official site <a href="http://www.nsthemes.com">NsThemes.com</a>.

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at report@nsthemes.com.*


== Installation ==

1. Unzip the .zip file
2. Upload the entire 'ns-search-products_free' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
